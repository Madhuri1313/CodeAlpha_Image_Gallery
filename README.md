# NovaFrame - Responsive Image Gallery

A modern and responsive image gallery built using HTML, CSS, and JavaScript.

## Features

- Responsive Grid Gallery
- Lightbox Image Preview
- Next / Previous Navigation
- Category Filters
- Hover Effects & Smooth Animations
- Keyboard Navigation
- Contact Form
- Mobile-Friendly Design
- Optimized Local Images

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

## Categories

- Nature
- Technology
- Animals
- Cars
- Travel

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Madhuri1313/CodeAlpha_Image_Gallery.git
```

2. Open the project folder

3. Open `index.html` in your browser

OR

4. Run using VS Code Live Server
